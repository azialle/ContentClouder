<p>
    <b>ContentClouder</b> is a Streamlit application that transforms YouTube transcripts into high‑resolution word clouds. It is designed to help users quickly summarize video content, highlighting key topics in a visually compelling way. It can be used for text analysis, research, or any presentations.
</p>

**Live Demo:** [**ContentClouder on Streamlit Cloud**](https://contentclouder.streamlit.app/)

## How It Works

- **Extract:** Paste a YouTube URL to retrieve the full transcript.
- **Visualize:** Generate a high-resolution word cloud highlighting key topics of the video.
- **Refine:** Customize themes, shapes, and filter keywords to visualize exactly what you need.
